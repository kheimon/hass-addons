# Kheimon's Home Assistant Add-Ons
Saving and/or mirroring some useful things for posterity.

## Installing the Repository
1. In Homeassistant navigate to **Settings** -> **Add-ons** -> **Add-on Store** -> **Repositories**
2. Add the repository: `https://github.com/kheimon/hass-addons`

## Add-ons in this repo
- [Acme.sh for Home Assistant](https://github.com/kheimon/hass-addons/tree/main/acme_sh): bash wrapper for [acme.sh](https://acme.sh) based off [acme.sh homeassistant addon](https://github.com/Angoll/acme.sh-homeassistant-addon) and intended to expose all supported DNS API settings.
- [acme.sHomeassistant](https://codeberg.org/hupf/homeassistant-addons/src/branch/main/acmeshomeassistant): Wrapper add-on for [acme.sh](https://acme.sh) written mostly in Python
